# Working Group Teaching Materials and Knowledge Base

The central element of Section EduTrain is a comprehensive offering of event formats and teaching materials on RDM and data literacy generally, covering both cross-disciplinary and discipline-specific needs in a target-group-appropriate way.

This includes:

- Slide sets for local multipliers
- Videos
- Hands-on formats
- Materials for in-person events
- Learning modules (e.g. online courses)
- Information on conference contributions, community meetups, newsletters, and helpdesks

To simplify the search for suitable materials, the Knowledge Base is linked to a **query wizard** whose queries are oriented to the properties of the target group profiles and offer filter options for material metadata. In the long term, a methodological-technological solution will be provided to present consortium content in a FAIR manner within NFDI.

## DALIA

The Knowledge Base is implemented through the **DALIA** project (Data Literacy Alliance), which serves as the shared platform for teaching materials across EduTrain and the NFDI.
