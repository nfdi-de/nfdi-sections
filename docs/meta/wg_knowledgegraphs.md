# Working Group Knowledge Graphs

Knowledge graphs are a key technology for implementing FAIR principles in data infrastructures and ensuring data interoperability both in automated data flows and for end users. The activities of the Knowledge Graphs working group aim to:

- Promote the use of knowledge graphs in NFDI consortia
- Enable cross-domain integration of distributed data in accordance with the FAIR principles
- Contribute to the joint development of tools and technologies that transform data into semantically usable knowledge across different knowledge domains
